:two:  ` How to Run this project`




1. Clone / Download this repository.

2. change directory to 'Server"

  `cd Server`
  
3. Install dependencies

  `npm install`
  
4. start Server

  `npm run dev`
  

  
5.Open another terminal

6.Change directory to Client

  `cd Client`
  
7.Install dependencies

  `npm install`
  
8.Run Client side

  `npm start`
